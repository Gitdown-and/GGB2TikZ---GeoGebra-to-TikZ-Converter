# GGB2TikZ---GeoGebra-to-TikZ-Converter
GGB2TikZ 是用HTML实现的，用于将 ggb文件 转换为 Tikz(PGF)代码。无需安装任何软件，直接在浏览器中完成转换，生成的代码可直接嵌入 LaTeX 文档。

## 使用方法
下载ggb2tikz.html，后用浏览器打开即可

## 功能
支持ggb中的大部分元素，除了半圆和圆弧什么的。这个工具仅支持2d绘图，对于3d没有做支持，因为懒，而且是ai写的。

1.可以处理ggb的颜色，和虚线，虚线仅支持一种类型，也就是说你换别的样式可能识别不了出来。

2.调整圆锥曲线的绘图限制，特别是双曲线和抛物线。

且绘图范围的形状是长方形。

3.可绘制单独的函数区间

单个函数区间表达方式"x1,x2"等于[x1,x2]，多个函数区间用"+"连接，比如"-1,0+1,2"。

函数曲线调整界面
![](函数曲线绘制界面.png)

这个界面默认是不显示的，只有识别出函数曲线时才会出现.

4.这个工具也对输出后的代码做了全屏查看的功能。

5.支持中文/English，在右上角切换。

##
注意ggb坐标系中的单位长度等于tikz中的1cm。
所以在绘制时应自行调整大小，对于函数也要缩放，当然缩放后的表达式并不难，所以也没做函数的缩放。

当然这个工具还有不足的地方，仅支持显函数，不支持隐函数，就连二次曲线也不行，所以要画隐函数只能用显函数一个个拼起来。
